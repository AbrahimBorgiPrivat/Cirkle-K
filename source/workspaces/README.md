# Power BI Workspace

This folder contains the PBIP workspace for the Circle K reporting case.

## Contents

- `pbip`
The Power BI project in PBIP format.

The main entry file is:

- `source/workspaces/pbip/Cirkle K Simulated Case.pbip`

## Related Assets

Shared visual assets are stored outside the PBIP folder under:

- `resource/powerbi/CirkleKPBITheme.json`
- `resource/powerbi/Images`

This keeps the report workspace separate from reusable images and theme files.

## Getting Started

1. Open `Cirkle K Simulated Case.pbip` in Power BI Desktop with PBIP support.
2. Point the semantic model to the PostgreSQL data generated by the project pipeline.
3. Apply or update the shared theme and image assets from `resource/powerbi` as needed.

## Notes

- The workspace is meant to be used with data produced by the services under `source/code/service`.
- The report project lives in source control, while generated data stays outside the PBIP folder.
