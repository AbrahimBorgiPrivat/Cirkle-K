# Power BI Workspace

This folder contains the PBIP workspaces for the Circle K interview and simulation cases.

## Contents

- `interview`
The interview PBIP project.
- `simulated`
The simulation PBIP project.

The main entry files are:

- `source/workspaces/interview/Circle K Interview.pbip`
- `source/workspaces/simulated/Cirkle K Simulated Case.pbip`

## Related Assets

Shared visual assets are stored outside the PBIP folder under:

- `resource/powerbi/interview`
- `resource/powerbi/simulation`

This keeps each report workspace separate from reusable images and theme files.

## Getting Started

1. Open the relevant `.pbip` file in Power BI Desktop with PBIP support.
2. Point the semantic model to the PostgreSQL data generated by the matching ETL flow.
3. Apply or update the shared theme and image assets from the matching `resource/powerbi/<case>` folder as needed.

## Notes

- The interview workspace is meant to be used with data produced by `service_interview_case1`.
- The simulation workspace is meant to be used with data produced by the JSON and simulation services under `source/code/service`.
- The report project lives in source control, while generated data stays outside the PBIP folder.
